# Modelo 145 App

## Instrucciones de instalación

1. Coloca la carpeta `modelo145_app` completa en tu PC/servidor.
2. Instala Node.js si no lo tienes.
3. En la carpeta `backend`, ejecuta:
   npm install
4. Inicia el servidor:
   node server.js
5. Abre el navegador y carga `index.html` desde `frontend/`.
6. Rellena los datos, firma y pulsa "Generar PDF".
7. El PDF se descargará listo para imprimir o enviar.
